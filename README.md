# Animal Sounds - Minimal Edition

A clean, minimal static website featuring 102 different animal sounds with a black, white, and grey color palette.

## Features

- 102 animals across 6 categories (Farm, Wild, Birds, Aquatic, Insects, Pets)
- Real MP3 audio playback with loop functionality
- Minimal black, white, and grey design
- Search and filter functionality
- Favorite system
- Statistics tracking
- Fully responsive design

## Setup

1. Open `index.html` in your browser
2. Add MP3 audio files to the `sounds` folder (see below)

## Adding Animal Sounds

To use real animal sounds, you need to add MP3 files to the `sounds` folder. The files should be named according to the animal name in lowercase with hyphens:

### File Naming Examples:
- `cow.mp3`
- `dog.mp3`
- `donkey.mp3`
- `guinea-pig.mp3` (for multi-word names)

### Where to Get Free Animal Sounds:

1. **Freesound.org** - https://freesound.org
   - Search for animal names
   - Filter by license (Creative Commons)
   - Download WAV/MP3 files

2. **Zapsplat.com** - https://www.zapsplat.com
   - Free sound effects library
   - Animal sounds category
   - Requires free account

3. **Soundbible.com** - http://soundbible.com
   - Public domain sounds
   - Animal category available
   - Direct MP3 downloads

4. **BBC Sound Effects** - https://sound-effects.bbcrewind.co.uk
   - Free for personal/educational use
   - High-quality recordings
   - Search by animal name

### Quick Start:

1. Create a `sounds` folder in the same directory as `index.html`
2. Download animal sound MP3 files
3. Rename them to match the animal names (lowercase, hyphens for spaces)
4. Refresh the page and click play!

## Fallback

If MP3 files are not found, the app will generate a simple tone as a fallback. The website will still work without the audio files, but you won't hear realistic animal sounds.

## Browser Compatibility

- Chrome/Edge: Full support
- Firefox: Full support
- Safari: Full support
- Mobile browsers: Full support

## Controls

- **Play/Pause**: Click the play button to start looping the sound, click again to pause
- **Favorite**: Click the heart icon to mark favorites
- **Search**: Type to filter animals by name or sound
- **Filter**: Click category tabs to filter by animal type
- **Details**: Click on any card to see more information
